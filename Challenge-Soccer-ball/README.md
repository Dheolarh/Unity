# Challenge Soccer Ball

Soccer-inspired challenge game focused on precision control, timing, and obstacle navigation.

## About the Game

Challenge Soccer Ball is a Sports Arcade project in my game development portfolio. The gameplay design emphasizes a simple core loop, responsive controls, and replay value.

## Technical Overview

- **Primary Stack:** C#, ShaderLab
- **Engine/Platform:** Unity (for Unity-based entries) and platform-specific tooling where applicable
- **Focus Areas:** Gameplay mechanics, scene setup, player interaction, balancing, and iteration
