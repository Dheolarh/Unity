# Barrier Dash

2D obstacle course game with increasing difficulty and reflex-based controls.

[![View Original Repository](https://img.shields.io/badge/View-Original%20Repository-181717?style=for-the-badge&logo=github)](https://github.com/Dheolarh/Barrier-Dash)

![Game Splash Screen](https://opengraph.githubassets.com/1/Dheolarh/Barrier-Dash)

## About the Game

Barrier Dash is a 2D Obstacle Runner project in my game development portfolio. The gameplay design emphasizes a simple core loop, responsive controls, and replay value.

## Technical Overview

- **Primary Stack:** C#, ShaderLab
- **Engine/Platform:** Unity (for Unity-based entries) and platform-specific tooling where applicable
- **Focus Areas:** Gameplay mechanics, scene setup, player interaction, balancing, and iteration
