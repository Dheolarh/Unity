## About the Game

Word Duel is a Multiplayer Word Game project in my game development portfolio. The gameplay design emphasizes a simple core loop, responsive controls, and replay value.

## Technical Overview

- **Primary Stack:** TypeScript, JavaScript, HTML, CSS
- **Engine/Platform:** Unity (for Unity-based entries) and platform-specific tooling where applicable
- **Focus Areas:** Gameplay mechanics, scene setup, player interaction, balancing, and iteration
