# Sumo Ball

Ball sumo wrestling on a floating island with physics-based push-and-survive gameplay.

[![View Original Repository](https://img.shields.io/badge/View-Original%20Repository-181717?style=for-the-badge&logo=github)](https://github.com/Dheolarh/Sumo-Ball)

![Game Splash Screen](https://opengraph.githubassets.com/1/Dheolarh/Sumo-Ball)

## About the Game

Sumo Ball is a Physics Arena project in my game development portfolio. The gameplay design emphasizes a simple core loop, responsive controls, and replay value.

## Technical Overview

- **Primary Stack:** C#
- **Engine/Platform:** Unity (for Unity-based entries) and platform-specific tooling where applicable
- **Focus Areas:** Gameplay mechanics, scene setup, player interaction, balancing, and iteration
